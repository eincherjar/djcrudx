# Przykład użycia DjCrudX

## Szybki start

```python\n# models.py\nfrom django.db import models\n\nclass Product(models.Model):\n    name = models.CharField(max_length=100)\n    price = models.DecimalField(max_digits=10, decimal_places=2)\n    category = models.ForeignKey('Category', on_delete=models.CASCADE)\n    is_active = models.BooleanField(default=True)\n    created_at = models.DateTimeField(auto_now_add=True)\n\nclass Category(models.Model):\n    name = models.CharField(max_length=50)\n    color = models.CharField(max_length=7, default='#ffffff')\n```\n\n```python\n# forms.py\nfrom django import forms\nfrom djcrudx.widgets import (\n    MultiSelectDropdownWidget,\n    ColoredSelectDropdownWidget,\n    DateTimePickerWidget,\n    ActiveStatusDropdownWidget,\n)\nfrom .models import Product\n\nclass ProductForm(forms.ModelForm):\n    class Meta:\n        model = Product\n        fields = '__all__'\n        widgets = {\n            'category': ColoredSelectDropdownWidget(),\n            'is_active': ActiveStatusDropdownWidget(),\n            'created_at': DateTimePickerWidget(),\n        }\n```\n\n```python\n# filters.py\nimport django_filters\nfrom djcrudx.widgets import DateRangePickerWidget\nfrom .models import Product\n\nclass ProductFilter(django_filters.FilterSet):\n    created_at = django_filters.DateFromToRangeFilter(\n        widget=DateRangePickerWidget(),\n        label=\"Data utworzenia\"\n    )\n    \n    class Meta:\n        model = Product\n        fields = ['name', 'category', 'is_active']\n```\n\n```python\n# views.py\nfrom djcrudx import create_crud\nfrom .models import Product\nfrom .forms import ProductForm\nfrom .filters import ProductFilter\n\n# Konfiguracja tabeli\nTABLE_CONFIG = [\n    {\n        \"label\": \"Nazwa\",\n        \"field\": \"name\",\n        \"value\": lambda obj: obj.name,\n        \"url\": lambda obj: (\"products:product_update\", {\"pk\": obj.pk}),\n    },\n    {\n        \"label\": \"Cena\",\n        \"field\": \"price\",\n        \"value\": lambda obj: f\"{obj.price} zł\",\n    },\n    {\n        \"label\": \"Status\",\n        \"field\": \"is_active\",\n        \"is_badge\": True,\n        \"badge_data\": lambda obj: [\n            {\n                \"name\": \"Aktywny\" if obj.is_active else \"Nieaktywny\",\n                \"background_color\": \"green-100\" if obj.is_active else \"red-100\",\n                \"text_color\": \"green-800\" if obj.is_active else \"red-800\"\n            }\n        ],\n    },\n]\n\n# Sekcje formularza\nFORM_SECTIONS = [\n    {\n        \"title\": \"Podstawowe informacje\",\n        \"columns\": 2,\n        \"fields\": [\"name\", \"price\"]\n    },\n    {\n        \"title\": \"Szczegóły\",\n        \"columns\": 1,\n        \"fields\": [\"category\", \"is_active\"]\n    }\n]\n\n# Tworzenie widoków CRUD\ncrud = create_crud(Product, ProductForm, ProductFilter)\n\nproduct_list = crud['list'](\n    TABLE_CONFIG,\n    page_title=\"Produkty\",\n    create_url=\"products:product_create\"\n)\n\nproduct_create = crud['create'](\n    FORM_SECTIONS,\n    page_title=\"Nowy produkt\"\n)\n\nproduct_update = crud['update'](\n    FORM_SECTIONS,\n    readonly_fields=[\"created_at\"],\n    page_title=\"Edytuj produkt\"\n)\n\nproduct_detail = crud['detail'](\n    FORM_SECTIONS,\n    page_title=\"Szczegóły produktu\"\n)\n\nproduct_delete = crud['delete'](\n    page_title=\"Usuń produkt\"\n)\n```\n\n```python\n# urls.py\nfrom django.urls import path\nfrom . import views\n\napp_name = 'products'\n\nurlpatterns = [\n    path('', views.product_list, name='product_list'),\n    path('create/', views.product_create, name='product_create'),\n    path('<int:pk>/edit/', views.product_update, name='product_update'),\n    path('<int:pk>/', views.product_detail, name='product_detail'),\n    path('<int:pk>/delete/', views.product_delete, name='product_delete'),\n]\n```\n\n## Dostępne widgety\n\n- `MultiSelectDropdownWidget` - Wielokrotny wybór z checkboxami\n- `SingleSelectDropdownWidget` - Pojedynczy wybór z radio buttonami\n- `ColoredSelectDropdownWidget` - Wybór z kolorowymi opcjami\n- `DateTimePickerWidget` - Wybór daty i czasu\n- `DateRangePickerWidget` - Wybór zakresu dat (dla filtrów)\n- `ActiveStatusDropdownWidget` - Status aktywności z czerwonym tłem dla \"Nie\"\n- `TextInputWidget` - Stylizowane pole tekstowe\n\n## Funkcje tabeli\n\n- **Sortowanie** - Kliknij nagłówki kolumn\n- **Filtrowanie** - Automatyczne formularze filtrów\n- **Paginacja** - Nawigacja między stronami\n- **Badges** - Kolorowe znaczniki w komórkach\n- **Linki** - Klikalne komórki z URL-ami\n- **Wyszukiwanie** - Wbudowana funkcjonalność wyszukiwania
